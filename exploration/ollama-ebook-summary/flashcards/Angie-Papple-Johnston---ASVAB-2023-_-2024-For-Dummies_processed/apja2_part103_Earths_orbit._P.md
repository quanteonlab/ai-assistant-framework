#### Principles of Electricity

Background context: The Electronics Information subtest covers the principles of electricity, including voltage (V), current (I), and resistance (R). Familiarize yourself with Ohm's Law and how these concepts apply to real-world scenarios.

:p What is Ohm's Law?
??x
Ohm's Law states that the current (I) flowing through a conductor between two points is directly proportional to the voltage (V) across the two points, and inversely proportional to the resistance (R).

Mathematically: $I = \frac{V}{R}$

Example problem: If the voltage across a resistor is 12V and its resistance is 4 ohms, what is the current flowing through it?
```java
public class OhmsLaw {
    public static void main(String[] args) {
        double voltage = 12; // in volts
        int resistance = 4; // in ohms
        double current = voltage / resistance;
        System.out.println("Current: " + current + " Amperes");
    }
}
```
