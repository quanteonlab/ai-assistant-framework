#### Order of Operations: PEMDAS
Mathematical equations with multiple steps need to be solved in a specific order. This is known as the order of operations, which can be remembered using the acronym PEMDAS (Please Excuse My Dear Aunt Sally).

:p What is the correct order of operations?
??x
The order of operations is:
1. Parentheses and other grouping symbols.
2. Exponents.
3. Multiplication and division from left to right.
4. Addition and subtraction from left to right.

For example, in the expression $3^2 + 3$:
- First, evaluate the exponent: $3^2 = 9$.
- Then perform the addition: $9 + 3 = 12$.

So, $3^2 + 3$ equals 12, not 53.
??x

```java
public class OrderOfOperationsExample {
    public static void main(String[] args) {
        int result = 3 * 2 + 3; // First multiplication then addition
        System.out.println(result); // Output: 9
    }
}
```
The code example demonstrates how to follow the order of operations.

#### Geometry Formulas for Area, Perimeter, and Circumference
When solving ASVAB math problems, you may need to calculate area, perimeter, or circumference. Here are some key formulas:

- **Area of a Rectangle**: $A = \text{Length} \times \text{Width}$- **Area of a Triangle**:$ A = \frac{\text{Base} \times \text{Height}}{2}$- **Area of a Circle**:$ A = \pi r^2$

For circles, the diameter equals 2 times the radius.

:p What is the formula for the area of a circle?
??x
The formula for the area of a circle is $A = \pi r^2 $, where $\pi $(pi) is approximately 3.14 and $ r$ is the radius.
??x

```java
public class CircleAreaExample {
    public static void main(String[] args) {
        double radius = 5;
        double area = Math.PI * Math.pow(radius, 2);
        System.out.println("The area of a circle with radius " + radius + " is: " + area); // Output: The area of a circle with radius 5 is: 78.53981633974483
    }
}
```
This code calculates the area of a circle given its radius.

#### Perimeter and Circumference Formulas

- **Perimeter of a Polygon**: The perimeter is found by adding up all side lengths.
  - For quadrilaterals (four-sided figures) or triangles, calculate the perimeter by summing the lengths of all sides.

- **Circumference of a Circle**: $C = \pi d $, where $ d $ is the diameter. Since the diameter equals $2r $, you can also use $ C = 2 \pi r$.

:p What is the formula for the circumference of a circle?
??x
The formula for the circumference of a circle is $C = \pi d $ or$C = 2 \pi r $, where$ d $is the diameter and$ r$ is the radius.
??x

```java
public class CircleCircumferenceExample {
    public static void main(String[] args) {

    }
}
```
This code calculates the circumference of a circle given its radius.
